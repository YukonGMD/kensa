# Kensa

Kensa is a graphical Pacman package review tool for Arch Linux.

## Download

You can download the latest AppImage from the [Releases](https://github.com/YukonGMD/kensa/releases/) page.

After downloading, mark it executable:

chmod +x Kensa-x86_64.AppImage
